# Hanako PDF
A brute force HTML to PDF script.

## Installation

```bash
npm install hanako-ts --save
npm install jspdf --save
```

## Usage
```javascript
import { HanakoPDF } from '...';

HanakoPDF.print(hanakoCollectionElement, options?);
```
